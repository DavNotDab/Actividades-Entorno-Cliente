¿En qué se diferencia JavaScript de Java?

Una diferencia entre Java y JavaScript son que Java es un lenguaje compilado, mientras que JavaScript es interpretado. Esto significa que se ejecuta línea a línea en lugar de todo de golpe después de compilar.

También se diferencian en la ejecución, ya que Java genera ejecutables pero JavaScript solo se ejecuta en el navegador.

Ej 10:

Describe las ventajas más importantes de usar JavaScript.

Una de sus mayores ventajas es su velocidad, ya que al ser un lenguaje script es muy ligero y rápido.

También es compatible con casi cualquier navegador y página web, por lo que es muy cómodo de usar.

A pesar de ser muy potente es sencillo, no es complejo de aprender y tiene muchas facilidades.

Es muy versátil, puede ser usado de muchas formas. Y más aún con la cantidad de frameworks que existen.
